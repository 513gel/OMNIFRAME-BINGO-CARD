OMNIFRAME MOBILE UPDATE

TO UPDATE YOUR EXISTING GITHUB PAGES SITE:
1. Extract this ZIP.
2. Open the existing GitHub repository.
3. Upload all extracted files to the root.
4. Allow GitHub to replace index.html.
5. Commit the changes.
6. Open the same GitHub Pages URL on your phone.
7. Refresh the page.

PHONE REVIEW:
- Review opens one undecided card at a time.
- Bottom left: Disapprove.
- Bottom middle: leave undecided and move later.
- Bottom right: Approve.
- Use the fixed bottom navigation to switch pages.
- Use Filters, progress & batch tools only when needed.
- Export Full Review Backup regularly.

Do not use private browsing and do not change the GitHub Pages URL.
