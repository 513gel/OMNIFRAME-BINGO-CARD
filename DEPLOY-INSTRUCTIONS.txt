OMNIFRAME PERSONALITY REVIEW v1.3

UPDATE THE GITHUB SITE
1. Export a Full Review Backup from the current site if you already made decisions.
2. Extract this ZIP.
3. Upload every extracted file and folder to the root of the existing GitHub repository.
4. Replace the old files.
5. Commit the update.
6. Wait for GitHub Pages to deploy.
7. Close the old browser tab.
8. Reopen the same Pages URL.
9. If the old version remains cached, append ?v=1.3 to the URL once.

NEW REVIEW FLOW
- The app presents one random undecided card at a time.
- Read “What this card is asking.”
- Read what approval and disapproval mean.
- Type an optional note directly on the card.
- Tap Approve or Disapprove. The decision and note save together.
- Tap Later to move the card to the back of the random queue.
- The bottom progress bar shows reviewed cards, remaining cards and percent complete.
- Decided cards do not reappear in the normal queue.

EXPORT
When finished, download:
- Full review backup
- Culled personality bible
- Rejection report

All three exports retain the notes attached to decisions.
